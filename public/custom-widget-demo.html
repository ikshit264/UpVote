🎨 Custom Widget Demo

See how to customize your UpVote widget with logo, FAQs, and product information

Complete Widget Configuration

This is a fully configured widget with custom logo, product overview, about text, and FAQs. Simply copy this code and customize it for your product:

<!-- Custom Widget Configuration --> <div class="upvote-widget" data-application-id="69a41f203a9a405a41b02afc" data-user-id="user_123" data-email="john@example.com" data-position="right" data-theme="light" <!-- Custom Logo (appears on button & header) --> data-logo-url="https://yourdomain.com/logo.png" <!-- Custom Product Overview --> data-product-overview="Our platform helps teams collaborate effectively and ship projects faster." <!-- Custom About Text --> data-about-text="We're a team of passionate developers building tools for modern teams." <!-- Custom FAQs (JSON array) --> data-faqs='[ { "question": "How does the free trial work?", "answer": "You get full access to all features for 14 days!" }, { "question": "Can I cancel anytime?", "answer": "Yes! Cancel anytime from your account settings." }, { "question": "Do you offer discounts?", "answer": "We offer 50% off for students and non-profits." } ]' ></div> <script src="http://localhost:3000/widget.js"></script>
💡 Pro Tip: The widget automatically detects configuration changes and updates in real-time!

✨ What You Can Customize

  • Custom Logo: Display your product logo on the widget button and in the FAQ header
  • Product Overview: Tell users what your product does and why feedback matters
  • About Section: Share information about your company or team
  • Custom FAQs: Add unlimited FAQs specific to your product
  • Position: Choose left or right side placement
  • Theme: Light or dark mode to match your brand

📝 FAQ JSON Format

Your FAQs must be formatted as a JSON array. Here's the structure:

data-faqs='[ { "question": "Your question here?", "answer": "Your answer here..." }, { "question": "Another question?", "answer": "Another answer..." } ]'
⚠️ Important: Make sure your JSON is valid. Use single quotes for the outer attribute and double quotes inside the JSON.

🖼️ Logo Requirements

Use square images (128x128px or larger). Supported formats: PNG, JPG, SVG, WebP. Always use HTTPS URLs.

📊 Best Practices

Keep FAQs concise (2-3 sentences). Write clear product descriptions. Test all configurations before deploying.

🔄 Dynamic Updates

Update data attributes via JavaScript and the widget will automatically refresh with new configuration.

🎯 Default Fallbacks

If no custom logo is provided, the default UpVote favicon is used. Default FAQs show when none are specified.
