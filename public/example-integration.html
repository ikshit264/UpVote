UpVote Widget Integration Example

This page demonstrates how to integrate the UpVote feedback widget into your website

Integration Steps

  1. Create an Account
    Sign up at /auth/signup and create your company account
  2. Get Your Company ID
    From your dashboard, copy the widget embedding code which includes your unique company ID
  3. Add the Widget
    Copy the two lines of code provided and paste them where you want the feedback widget to appear
  4. Test & Deploy
    Test the widget locally, then deploy to production when ready
Widget HTML Code
Replace YOUR_COMPANY_ID with your actual company ID:
<div data-upvote-company-id="YOUR_COMPANY_ID"></div> <script src="http://localhost:3000/widget.js"></script>
Production Domain
When deploying to production, update the script src to your actual domain:
<div data-upvote-company-id="YOUR_COMPANY_ID"></div> <script src="https://your-domain.com/widget.js"></script>
Optional User Tracking
Track specific users by adding a user ID attribute:
<div data-upvote-company-id="YOUR_COMPANY_ID" data-upvote-user-id="user@example.com"></div> <script src="https://your-domain.com/widget.js"></script>
